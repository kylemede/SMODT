void MCMCorbFuncObj::simulator ( )

This is the core function that performs the MCMC process for a single chain of a multi or single chain/thread/process MCMC simulation started by the Python file BinaryOrbSimStarterDuo.py and controlled with the MCMC_ProcessManagerDuo.
